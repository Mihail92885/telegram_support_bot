# Telegram Support Bot

Бот технической поддержки для тестового задания.

## Функции
- Команды `/start` и `/help`
- Распознаёт ключевые слова: принтер, интернет, ККМ
- Даёт простые рекомендации

## Как запустить
1. Установи зависимости: `pip install -r requirements.txt`
2. Получи токен у [@BotFather](https://t.me/BotFather)
3. Вставь токен в файл `bot.py`
4. Запусти: `python bot.py`

## Процесс работы с Claude Code
Я попросил Claude Code создать Telegram-бота техподдержки на aiogram. Claude сгенерировал базовую структуру с командами /start и /help. Я вручную добавил логику ответов по ключевым словам (принтер, интернет, ККМ). Код запустился с первого раза, тестирование прошло успешно. Весь процесс занял около 20 минут.
